# Image Card

```js
import { Card, CardImage } from "tropical-components-react/Card";
```

### Usage

```html
<Card
    image={
        <CardImage
            posterImage="https://static.ssbintel.com/assets/portals/ssb/background/football_background.jpg"
            title="Hello, World!"
            size="medium"
        />
    }
>
    <div>
        <p>This is a card with an image</p>
    </div>
</Card>
```
